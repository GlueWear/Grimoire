# GRIMOIRE UI Design Spec

This document outlines UI changes for the GRIMOIRE chat application. Use this as the reference for all visual updates.

---

## Design Principles

- **Editor-inspired aesthetic** - Clean, minimal, functional like a code editor
- **Cyberpunk undertones** - Dark theme, but refined and readable
- **Desktop-focused** - No mobile considerations for now
- **Readability first** - Larger text, no glow effects, clear contrast

---

## Color Palette

Replace ALL existing color usage with these new values:

| Role | Old | New Hex | Usage |
|------|-----|---------|-------|
| Urbit ID | amber/orange | `#CE9178` | Urbit usernames, Urbit-related UI elements |
| Visitor | blue | `#569CD6` | Visitor usernames, visitor-related UI elements |
| Bot/Agent | green | `#6A9955` | Bot usernames, notification dots, bot-related UI |

**Rule:** Anywhere the old amber/orange/blue/green appears, replace with the corresponding new color. This includes text, borders, backgrounds, dots, highlights - everything.

---

## Typography

### Text Size Increase

The current text is too small compared to Twitter, Tlon, Reddit, etc.

- **Increase base font size** - Main message text needs to be larger
- **Scale proportionally** - All other text (headers, labels, timestamps, wallet, modals) scales up proportionally
- **Login screen is baseline** - The login screen text size is correct; use it as reference
- **Exception:** Urbit name selection buttons on login are too small - increase those

---

## Effects to Remove

### Scan Lines
- Remove ALL scan line effects throughout the app
- No CRT/retro scan line overlays anywhere

### Glow
- Remove ALL text glow effects
- Remove any box glow/shadow effects that create a "glow" look
- Standard subtle shadows for depth are fine, but no glowing

---

## User Avatars (PFP)

Add avatar/profile picture squares throughout the app.

### Spec:
- **Shape:** Square with slightly rounded corners
- **Locations:** User list sidebar AND next to messages in chat
- **Size:** Proportional to text size (roughly 1.5-2x line height)

### Default avatars by user type:

**Urbit users (`~name`):**
- Display their Urbit sigil
- Use `urbit-ob` library to generate
- Sigil should fit within the square

**Visitors (`@name`):**
- First letter of username
- Background: `#569CD6` (visitor blue)
- Letter: white or contrasting color

**Bots (`%name`):**
- First letter of bot name
- Background: `#6A9955` (bot green)
- Letter: white or contrasting color

---

## Message Grouping

Consecutive messages from the same user should be grouped.

### Behavior:
- First message in a group shows: PFP + username + timestamp + message
- Subsequent consecutive messages from same user show: just the message (indented to align with first message's text)
- No repeat of PFP or username until a different user sends a message

### Example:
```
[sigil] ~disden-talhes                    2:34 PM
        Hey everyone
        Just pushed some updates
        Let me know what you think

[D]     @visitor-dan                       2:35 PM
        Looks good!

[sigil] ~disden-talhes                    2:35 PM
        Thanks!
```

## @Mentions in Message Bar

When typing an @mention in the message input:

- After the user types `@username` or `@~planet-name` or `@%botname`
- The mention text should highlight in the appropriate color:
  - Urbit names: `#CE9178`
  - Visitor names: `#569CD6`
  - Bot names: `#6A9955`
- This gives visual feedback that the mention was recognized

## Artifacts

### Remove Borders
- No border lines around artifacts
- Artifacts should feel embedded in the space, not boxed

### Controls
- Keep existing controls in top right corner
- Keep hover-to-reveal behavior for chat message controls
- No changes needed to canvas artifact functionality

### Scrollbars
- Subtle, thin scrollbars
- Should not be visually prominent
- Consider auto-hide when not scrolling

## Implementation Notes

### Suggested CSS Variables

```css
:root {
  /* User type colors */
  --color-urbit: #CE9178;
  --color-visitor: #569CD6;
  --color-bot: #6A9955;
  
  /* Base font size - adjust this to scale everything */
  --font-size-base: 16px; /* or larger */
  --font-size-small: 0.875em;
  --font-size-large: 1.125em;
}
```

### Sigil Generation

The `urbit-ob-browser.js` file is already included. For sigil rendering, you may need to add `urbit-sigil-js` or render sigils as SVG. The sigil is derived from the `@p` (planet name).

### Message Grouping Logic

When rendering messages:
1. Track the previous message's author
2. If current message author === previous message author, render without PFP/name
3. Reset when author changes
